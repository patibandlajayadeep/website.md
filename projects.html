Healthcare & Order-Driven Projects

Real examples of how I’ve improved claims accuracy, reduced manual work, and made EHR workflows easier for billing staff and SMEs.

Claims & Pricing

Claims Rules Validation Studio

A comprehensive studio for analysts, QA, and engineering teams to build, test, and validate claim pricing rules with real-time visibility into rule outcomes and pricing behavior — without needing SQL or production access.

  • Built a UI that highlights rule fires, misses, and pricing changes with expected vs. actual comparisons.
  • Added batch validation mode to test hundreds of claims and produce pass/fail accuracy reports for QA review.
  • Integrated detailed audit trails including inputs, outputs, reason codes, and timestamps for compliance and debugging.
  • Enabled exporting of test cases and validation reports for analysts, developers, and QA teams.
Orders & EHR Workflow

Order Journey Explorer

An interactive tool that lets operations, billing, and support teams trace a patient order end-to-end across labs, radiology, eligibility, pricing, and claims systems — without needing to query databases or sift through logs.

  • Designed an end-to-end order timeline that stitches events from orders, scheduling, billing, and claims into a single view.
  • Built backend aggregation APIs in .NET Core to join SQL data, message logs, and third-party services into time-ordered journeys.
  • Implemented filters and search by order ID, MRN, payer, facility, provider, status, and date range to quickly locate problem orders.
  • Added drill-down views for rule hits, denials, and resubmissions with exact reason codes and timestamps.
  • Instrumented telemetry and logging (latency, error spikes, retries) to help engineering debug production issues faster.
Vendor Apps

Vendor Configuration Portal

A secure internal web application that allows business teams to configure payers, fee schedules, vendors, reimbursement settings, and workflow rules—all without requiring direct database access.

  • Implemented role-based access and fine-grained auditing with .NET Core and Azure AD.
  • Implemented an approval workflow for configuration changes with reviewers, email notifications, and full audit history.
  • Added detailed auditing and activity logs, improving compliance and traceability for financial and claim-processing changes.
  • Reduced manual DBA “DB change” tickets by more than half, speeding up releases.
  • Improved data integrity by validating configuration changes before they reach production.
